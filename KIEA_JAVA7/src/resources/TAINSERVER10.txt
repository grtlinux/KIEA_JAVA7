[ Windows 2008 R2 Installation ]

	1. BIOS > Boot
		
		> Standard Boot Order

			1st Boot Device    [ Hard Drive ]
			2nd Boot Device    [ Disabled ]
		
		> Hard Disk Drives

			1st Drive   [ HDD: WDC WD3200K ]
			2nd Drive   [ ..... ]
		
		> USB Device Boot Priority  [ Low ]    <--  [ High ] : for USB install
	
	
	2. LAN
	
		> control netconnections
		
			#1 : 192.168.0.10
			#2 : 192.168.1.10

	
	3. 사용자 변경
	
		> control userpasswords
		
			계정이름변경: Administrator --> taincokr
	
	
	4. 시스템 속성
	
		> sysdm.cpl
		
			컴퓨터 이름
				작업 그룹:   TAINGROUP
				컴퓨터 이름: TAINSERVER10
			
			원격                 -> 3389 port변경 필요
				원격 데스크톱
					- 네트워크 수준 인증을 사용하여 원격 데스크톱을 실행하는 컴퓨터에서만 연결 허용(보안 수준 높음)
	
	5.

[ 공유폴더 삭제 ]

	1. 확인
	
		DOS> net share
		
			공유 이름   리소스                        설명
			
			-------------------------------------------------------------------------------
			ADMIN$       C:\Windows                      원격 관리
			C$           C:\                             기본 공유
			D$           D:\                             기본 공유
			IPC$                                         원격 IPC
			O$           O:\                             기본 공유
			P$           P:\                             기본 공유
			R$           R:\                             기본 공유
			S$           S:\                             기본 공유
			T$           T:\                             기본 공유
			명령을 잘 실행했습니다.	
		
	
	2. ADMIN$, C$.... 삭제
		
		DOS> reg.exe query "HKLM\SYSTEM\CurrentControlSet\Services\LanmanServer\Parameters"        <- 확인
		
		DOS> reg.exe add   "HKLM\SYSTEM\CurrentControlSet\Services\LanmanServer\Parameters" /v AutoshareWks    /t REG_DWORD /d 0 /f   <- 생성(win7)
		DOS> reg.exe add   "HKLM\SYSTEM\CurrentControlSet\Services\LanmanServer\Parameters" /v AutoshareServer /t REG_DWORD /d 0 /f   <- 생성(win2008)


	3. IPC$.... 삭제

		DOS> reg.exe query "HKLM\SYSTEM\CurrentControlSet\Control\Lsa"        <- 확인
		
		DOS> reg.exe add   "HKLM\SYSTEM\CurrentControlSet\Control\Lsa" /v RestrictAnonymous /t REG_DWORD /d 2 /f      <- 변경 (0 -> 2)


	4. 제거
	
		DOS> net share ADMIN$ /delete

		
	5. Reboot
