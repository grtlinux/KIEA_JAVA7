[ Server-Linux, Client-Windows ]

1. Linux에 삼바 서버를 설치한다.

	$ sudo apt-get install samba samba-common-bin
	
2. Linux 삼바서버 중단

	$ sudo service samba stop
	
3. Linux 삼바서버 환경파일을 수정

	$ sudo vi /etc/samba/smb.conf -> vi에서 라인번호를 보려면 ':set nu'
	
		102라인: security = user
		250라인: read only = no
		
4. kang 계정을 만든다. 비밀번호는 필요하면 입력한다.

	$ sudo smbpasswd -a kang
	
5. Linux 삼바서버 실행

	$ sudo service smb start
	
6. Windows에서 네트워크 드라이브 연결

	DOS> net use S: \\[ip or hostname]\[shared name]

7. 네트워크 드라이브 제거

	DOS> net use S: /delete
